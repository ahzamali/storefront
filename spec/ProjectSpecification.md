# Project Specification

This project is about creating a system which will be used to manage the inventory of a store.The system will be used to add, remove, update and view the inventory of the store. The system will have a backend system which is hosted on cloud and a mobile device app which can be installed on a android device.

The backend system will have the following features:
- User Authentication
- Inventory Management
- Order Management
- Report Generation
- User Management
- Product Management
- Category Management
- Sub Category Management
- Brand Management
- Unit Management
- Unit Conversion Management
- Price Management
- Stock Management
- Order Management
- Report Generation
- User Management
- Notification Management
- invoce generation 

The mobile app will have the following features:
- User Authentication
- Inventory Management
- Order Management
- Report Generation
- User Management
- Notification Management
- invoce generation 
- invetory scanning using books barcode/ISBN

We will support adhoc stores, where an inventory is allocated to a virtual store, managed by an employee. 
We should be able to generate overall report from this virual store for the sale amount. 
We are initally targetting sale of books, which will be done in bundles. The virtual stores should have the capabilties to sell books in budles with exclusions.
The final report should contain information about what all was sold and the remaining inventory from the virtual store.
